🐾 PANTHER HOLMES OSINT v5.0

Termux, Linux ve Windows üzerinde çalışan gelişmiş bir OSINT (Open Source Intelligence) aracıdır.

🚀 Özellikler

- Telegram kullanıcı adı analizi
- GitHub profil bilgileri
- Domain ve IP analizi
- DNS sorguları
- Email analiz araçları
- Telefon numarası yardımcı araçları
- Çoklu platform kullanıcı adı kontrolü
- Renkli Terminal Arayüzü (CLI)
- JSON ve TXT rapor çıktıları

📦 Gereksinimler

Python 3.9+

Gerekli kütüphaneler:

- requests
- colorama
- dnspython

📱 Termux Kurulumu

pkg update && pkg upgrade -y
pkg install python git -y

git clone https://github.com/pantixm/Panter-Holmes.git

cd Panter-Holmes

pip install -r requirements.txt

python panter.py

🐧 Linux Kurulumu

sudo apt update

sudo apt install python3 python3-pip git -y

git clone https://github.com/pantixm/Panter-Holmes.git

cd Panter-Holmes

pip3 install -r requirements.txt

python3 panter.py

🪟 Windows Kurulumu

1. Python'u yükleyin.
2. CMD veya PowerShell açın.

git clone https://github.com/pantixm/Panter-Holmes.git

cd Panter-Holmes

pip install -r requirements.txt

python panter.py

requests
colorama
dnspython
